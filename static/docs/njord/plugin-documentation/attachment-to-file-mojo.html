njord:attachment-to-file

Full name:

eu.maveniverse.maven.plugins:njord:0.9.0:attachment-to-file

Description:

Writes out attachment of the given store to a file.

Attributes:

  • Executes as an aggregator goal.
  • The goal is thread-safe and supports parallel builds.

Required Parameters

Name Type Since Description
<attachmentFile> File - The file to write attachment to. Mandatory parameter. If file exists, will be overwritten.
User Property: njord.attachmentFile
<store> String - The name of the store to write out attachment from. Mandatory parameter.
User Property: njord.store

Optional Parameters

Name Type Since Description
<attachmentName> String - The attachment name. Optional parameter, if not present, attachment file name will be used.
User Property: njord.attachmentName
<skip> boolean - Plugin configuration to skip the Mojo. The Mojo can also be skipped by user property $mojoName.skip or $prefix.$mojoName.skip as well (check specific Mojo).
Default: false

Parameter Details

<attachmentFile>

The file to write attachment to. Mandatory parameter. If file exists, will be overwritten.
  • Type: java.io.File
  • Required: Yes
  • User Property: njord.attachmentFile

<attachmentName>

The attachment name. Optional parameter, if not present, attachment file name will be used.
  • Type: java.lang.String
  • Required: No
  • User Property: njord.attachmentName

<skip>

Plugin configuration to skip the Mojo. The Mojo can also be skipped by user property $mojoName.skip or $prefix.$mojoName.skip as well (check specific Mojo).
  • Type: boolean
  • Required: No
  • Default: false

<store>

The name of the store to write out attachment from. Mandatory parameter.
  • Type: java.lang.String
  • Required: Yes
  • User Property: njord.store
